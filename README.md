# Upload Labs Save Editor
 
A browser-based save editor for Upload Labs. I made this because I couldn't find any, so here it is.

**[Open the Editor](https://voidpossum.github.io/Upload-Labs-Save-Editor/)**

## How to use
 
1. Open the editor using the link above
2. Load your `savegame.dat` file
3. Edit what you want (tokens, money, research...)
4. Export and replace your save file

## Save file location
 
**Windows:**
```
C:\Users\%username%\AppData\Roaming\Upload Labs\savegame.dat
```
**Linux:**
```
~/.local/share/Upload Labs/savegame.dat
```
